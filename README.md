# InfluencerHub Client

A React-based frontend application for the InfluencerHub platform, providing an interactive interface for managing influencer profiles, campaigns, and analytics. Features a polished UI with animated interactions and responsive data visualizations.

## Tech Stack

- **React** ^18.2.0 with Create React App
- **React Router** v6 for client-side routing
- **Ant Design** 4 component library with icons
- **Tailwind CSS** for utility-first styling
- **Recharts** for data visualization and analytics charts
- **React Slick** for carousel components
- **React Animated Cursor** for custom cursor effects
- **Video React** for video playback

## Features

- Influencer profile browsing and management
- Campaign tracking and analytics dashboards
- Interactive data visualizations with charts
- Responsive design with mobile-first approach
- Animated UI transitions and custom cursor effects
- Video content integration
- Carousel/slider components for content showcase

## Getting Started

### Prerequisites

- Node.js >= 14
- npm or Yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mhmalvi/influencerhub-client.git
cd influencerhub-client

# Install dependencies
npm install
# or
yarn install
```

### Development

```bash
npm start
```

### Build

```bash
npm run build
```

### Testing

```bash
npm test
```
